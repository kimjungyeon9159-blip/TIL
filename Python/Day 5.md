### (과제1)


##### 문제

주어진 문자열에서 특정 문자의 개수를 세는 count_character 함수를 작성하시오. 

문자열과 대상 문자를 인자로 받아 개수를 반환해야 한다.

##### 실행결과
				
2




```

def count_character(sentence,char):
    """주어진 문자열에서 특정 문자의 개수를 세어 반환하는 함수입니다.
    >>> count_character(hello)
    1
    """
    return sentence.count(char)

result = count_character("Hello, World!", "o")
print(result)  # 2
```

### 주의점

1) return 작성 할 때 .count 앞에 함수 적용 대상인 객체를 적어 주어야한다
2) 매개변수를 작성할 때,  return에 적용할 객체와 반환값으로 필요한 객체 char을 모두 () 안에 작성하는 것으로 이해해보자

"""
[정답 및 해설] 내장 함수 없이 최댓값 / 최솟값 구하기

핵심 아이디어
  '지금까지 본 것 중 가장 큰 값'을 기억하는 변수를 하나 두고,
  새 값을 만날 때마다 기억하던 값과 비교해서 갱신한다.

비유
  운동장에서 키가 제일 큰 학생을 찾을 때, 전체를 줄 세우지(정렬) 않아도
  '지금까지 만난 애들 중 제일 큰 애'를 옆에 두고 한 명씩 비교하며
  걸어가면 한 바퀴만에 찾을 수 있다.
"""

numbers = [3, 1, 4, 1, 5, 9, 2, 6]

# [1] 초기값 설정이 이 문제의 핵심 함정이다.
#     0으로 초기화하면 원소가 모두 음수일 때([-5, -1, -3])
#     최댓값이 0으로 잘못 나온다. 반드시 '실제 데이터의 첫 원소'로 시작한다.
minimum = numbers[0]
maximum = numbers[0]

# [2] 첫 원소는 이미 기준으로 썼으므로 인덱스 1번부터 순회한다.
#     numbers 전체를 순회해도 결과는 같지만(첫 비교가 자기 자신),
#     불필요한 1회 비교를 줄이는 것이 정석이다.
for number in numbers[1:]:
    # [3] 지금 값이 기록보다 작으면 최솟값 기록을 교체한다.
    if number < minimum:
        minimum = number

    # [4] 한 값이 min이면서 동시에 max일 수는 없으므로 elif를 써도 정답이다.
    #     여기서는 두 조건이 논리적으로 독립임을 보여주기 위해 if를 사용했다.
    if number > maximum:
        maximum = number

print('최솟값:', minimum)
print('최댓값:', maximum)


# ---------------------------------------------------------------
# [참고] 자주 나오는 오답 유형
#
#   (1) minimum = 0, maximum = 0 으로 초기화
#       => 전부 음수인 데이터에서 오답.
#
#   (2) minimum = 9999, maximum = -9999 같은 '매직 넘버' 사용
#       => 데이터 범위를 벗어나면 즉시 깨진다.
#          첫 원소로 초기화하는 방식이 더 견고하다.
#
#   (3) 최댓값을 먼저 찾고 리스트를 다시 순회해 최솟값을 찾기
#       => 정답이지만 리스트를 2번 훑는다. 위 코드는 1번만 훑는다.
