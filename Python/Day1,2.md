
# 리스트 활용하기
## 주의할 것
  many_zero_list 변수에 숫자 0을 25만개 가지고 있는 리스트를 할당한다, 리스트와 곱셈 연산자를 활용하여 할당한다.
  
  many_zero_list 의 길이를 출력한다. 

### 정답
```

many_zero_list=('0'*250000)

print(len(many_zero_list))

* 숫자0을 25만개 가지고 있는 리스트와 그 길이 출력 *

```


# 기억해야할 것

   ## 1. 코드가 아니라 주석을 먼저 작성할 것, 주석으로 시나리오를 먼저 작성한 후 코드를 짜야함.

   #### 무엇을 출력하고 있는지 간단하게라도 포인트를 적어주는 것이 필요

   #### 알고리즘 문제를 풀 때 중요함

   ## 2. 코드 결과를 상상하지 말기

   #### 무조건 실행 후 결과를 보고, 에러를 많이 내면서 수정해가는 것이 중요

   #### print() 계속 쓰면서 코드 작성해가는 것이 중요

   ## 3. 계속 변화를 보면서, 값을 누적해가면서 확인하는 습관을 들이는 것이 중요 (print를 한줄 한줄 찍어가며 확인하는 것이 중요)

   #### 숫자가 커서 확인하기 어려우면, 작은 수 일때 부터 확인하는 것으로 시작하기.
